# Frontend (React + Vite)
Interface CRM (React + Vite) connectée à l’API NestJS.

## Prérequis
- Node.js 20+

## Configuration
Dans `.env.development`, vérifier:

```
VITE_API_URL=http://localhost:3001
```

## Démarrage
```bash
npm install
npm run dev
```

App: `http://localhost:5173` (ou un autre port si 5173 est occupé).

## Scripts utiles
- `npm run lint`
- `npm run build`
